Щоб сайт почав працювати, потрібно:
  В папці Backend:
    1. Відкрити її в Visial Studio Code
    2. Створити термінал
       3. Написати команди:
    4. py -m venv venv
    5. .\venv\Scripts\activate
    6. pip install -r req.txt
    7. cd ./server
    8. pip install django-filter
    9. pip install mysqlclient
    10. py manage.py runserver

  В папці Frontend:
    1. Відкрити її в Visial Studio Code
    2. Створити термінал
       3. Написати команди:
    4. npm i
    5. npm run dev
